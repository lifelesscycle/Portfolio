§ 02

Notices & Capabilities

Planning

One question becomes several

The Planner reads your question and stakes out the sub-inquiries a real researcher would chase first — no single-shot guesswork, no missed angles.

Retrieval-Augmented Generation

Evidence before opinion

Every answer is grounded in retrieved sources — internal documents, a knowledge graph, and the open web — before a single sentence is written.

Citation Verification

Every claim, checked against its source

A dedicated Verifier agent re-reads each claim against the document it came from and flags anything that doesn't hold up — nothing ships unchecked.

Hybrid LLM Support

Cloud or local, your call

Run the committee on cloud LLMs for scale, or on Ollama locally for privacy and cost control — the pipeline doesn't care which model does the reading.

Distributed Workflows

Built to not fall over

Celery workers and Redis queues run each agent's task independently, so a slow search or a stalled verifier never stalls the whole report.

Structured Reports

Research, ready to read

Findings land as a proper report — introduction, key findings, evidence, conclusion — not a wall of chat bubbles you have to reassemble yourself.
